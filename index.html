Tags para desenvolvimento básico de um HTML

(Voltar)

Tags

    h1 à h6 - São ferramentas de formatação de texto.

    ul - Define uma lista não ordenada.

    li Define uma lista ordenada.

    small - Define uma fonte menor dentro de um texto.

    a - Usada junto com o atributo "href=", serve para designar um lugar.

    href - Usada junto com a tag "a", indica uma âncora com outro documento.

    i - Define o texto como Itálico.

    strong - Define a palavra como forte, deixando ela em negrito.

    b - Assim como "Strong", deixa o texto em negrito, mas sem muita relevância.

    mark - Faz uma marcação de destaque no texto.

    ol - Utilizado para iniciar uma lista.

    p - Define o inicio de um paragráfo.

    u - Deixa o texto sublinhado.

    blockquote - Serve para definir uma citação.

    font - Define a fonte que será utilizada no texto.

    del - Representa uma parte do texto que foi excluído.

    abbr - Usada para definir uma abreviação dentro do texto.

    id - Define uma identidade para atribuir um elemento a um lugar específico.

    sub - Define uma palavra subscrita ao texto na mesma linha.

    sup - Define uma pavalra abaixo do texto mas na mesma linha.

    hr - Define uma linha temática na página.

Listas ordenadas e não ordenadas

(Voltar)

As listas ordenadas ou numeradas são usadas para indicar alguma sequência ou numeração. As tags usadas para criar uma lista ordenada são 'ol' e 'li'. A tag 'ol' é usada para definir a lista e a tag 'li' é usada para cada item da lista.

As listas não numeradas são usadas para listar itens, sem se preocupar com sua sequência. Chamamos de lista de marcadores apenas. As tags usadas para criar uma lista não ordenada são 'ul' e 'li'. A tag 'ul' é usada para definir a lista e a tag 'li' é usada para cada item da lista.

Inserindo links e imagens em seu html

(Voltar)

Para que seja feita a inserção de links e imagens podemos utilizar as tags "a=href=" ou "img" juntamente com o atributo 'scr'.

Ao utilizar de imagens que estão dentro de pastas precisamos indicar o local corretamente e o nome da imagem como no exemplo:

- img scr= "imagens/imagem.jpg"

Quando utilizamos imagens com link direto da internet ficamos dependentes do dominio, caso essa imagem seja excluida da internet, ela sairá também do nosso site.

Da mesma forma acontece com um link de outro site, se for excluido, o link de dentro da nossa pagína não funcionará mais.
